# Agence test de competencias.

## Docker basado en eggjs-mysql-docker
### https://github.com/ycjcl868/eggjs-mysql-docker

## Requirements

- docker
- docker-compose

## Deploy && Usage

```sh
# start
$ docker-compose up -d

# stop
$ docker-compose down

# remove
$ docker-compose down -v
```
